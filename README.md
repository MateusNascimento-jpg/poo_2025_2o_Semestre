# Fundamentos de Programação Orientada a Objetos | Java

Este repositório concentra os projetos práticos desenvolvidos para solidificar a transição da lógica estruturada para o paradigma de Orientação a Objetos (POO). O foco central consistiu em construir a base técnica e arquitetural de software, servindo como o alicerce preparatório essencial para o posterior desenvolvimento de APIs e sistemas web complexos.

## O que foi desenvolvido?
O escopo abrange a criação de sistemas de registo e a manipulação de dados operando inteiramente em memória. A evolução do código demonstra a interação progressiva entre múltiplas classes, simulando regras de negócio reais através da troca de mensagens e colaboração entre objetos, isolando a complexidade de integrações externas para focar puramente na arquitetura do código.

## Tecnologias e Conceitos Aplicados
O desenvolvimento priorizou a organização estrutural, a modularidade e a reutilização de componentes. As principais competências consolidadas incluem:

* **Pilares da POO:** Implementação prática e aprofundada da modelagem de classes, instanciação de objetos, encapsulamento, herança, polimorfismo e abstração.
* **Gestão de Dados em Memória:** Utilização intensiva do Java Collections Framework para o armazenamento, pesquisa e gestão dinâmica de múltiplos objetos durante a execução.
* **Modelagem de Domínio:** Tradução de requisitos lógicos em modelos orientados a objetos, garantindo a separação de responsabilidades e o baixo acoplamento entre as partes do sistema.
* **Qualidade e Documentação de Código:** Foco na legibilidade e manutenção do software, aplicando boas práticas de estruturação e comentários técnicos que detalham a responsabilidade e o comportamento de cada classe e método.
